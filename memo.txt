!!!!!! CAUTION !!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
matplotlibを表示させるためにはWSL内からdockerを起動させること
!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!

pip freeze > requirements.txt
でライブラリインストール用のテキストファイルを作成する

コンテナの起動
docker compose up -d

コンテナのシャットダウン
docker compose down

コンテナ内に入るコマンド（numerical_calculation_pythonはコンテナ名）
docker container exec -it numerical_calculation_python bash
